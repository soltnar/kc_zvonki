# КЦ звонки

Статический сайт для анализа Excel-выгрузок колл-центра:

- общая статистика звонков;
- статистика по операторам;
- входящие, исходящие и пропущенные;
- нагрузка по часам, дням недели и месяцам;
- тепловая карта день/час;
- расчет рекомендуемого количества операторов по часам;
- детализация строк `Сбис` через отдельный файл СБИС.

## Локальный запуск

Откройте папку проекта и запустите простой сервер:

```bash
python3 -m http.server 8080
```

После этого откройте:

```text
http://localhost:8080
```

## GitHub Pages

Репозиторий можно публиковать как обычный Pages-сайт из ветки `main`, папка `/root`.

Сайт не требует backend-сервера: Excel-файлы читаются в браузере, данные не отправляются наружу.
